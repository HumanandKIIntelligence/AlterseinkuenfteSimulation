# Alterseinkünfte Simulation

Streamlit-App zur Simulation von Renteneinkünften inkl. Steuer & Krankenversicherung.
Läuft auf **http://localhost:8502**

## Starten

```bash
docker compose up -d
```

## Stoppen

```bash
docker compose down
```

## Neu bauen (nach Änderungen an requirements.txt oder Dockerfile)

```bash
docker compose up --build -d
```

## Logs anzeigen

```bash
docker compose logs -f
```
